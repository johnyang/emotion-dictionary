# Emotion Dictionary 情绪词典

基于《情绪词典：你的感受试图告诉你什么》的情绪分析工具。

## 🚀 快速开始

### 查询情绪
```bash
node scripts/query.js query 焦虑
node scripts/query.js query 快乐
```

### 列出所有情绪
```bash
node scripts/query.js list
```

### 搜索情绪
```bash
node scripts/query.js search 不安
```

### 分析事件
```bash
node scripts/analyze.js analyze "今天工作压力很大"
```

## 📦 功能特点

- **32 个核心情绪** - 覆盖 7 个感受层次
- **快速查询** - query/list/search 三种模式
- **事件分析** - 4 步分析流程 + 回归疗法视角
- **中文友好** - 完整中文支持，拼音搜索

## 📚 情绪层次

1. 原始感受 - 身体最直接的感受
2. 基本情绪 - 喜、怒、哀、惧、惊、厌恶
3. 情绪 - 焦虑、抑郁、孤独等
4. 情感 - 爱、感恩、信任、希望等
5. 公情 - 安全感、归属感、成就感等
6. 共情性情感 - 同理心、怜悯等
7. 超越性情感 - 意义感、使命感、存在感等
