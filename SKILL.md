# Emotion Dictionary (情绪词典)

基于《情绪词典：你的感受试图告诉你什么》的情绪分析工具，帮助你理解感受背后的心理意义。

## 功能

- 查询情绪的完整信息（定义、身体反应、心理意义、应对策略）
- 列出所有情绪（按分类）
- 搜索关键词匹配的情绪
- 分析事件中的情绪并给出建议

## 安装

本技能已内置在 OpenClaw 中，无需额外安装。

## 使用示例

```bash
# 查询特定情绪
emotion-dictionary query 焦虑
emotion-dictionary query 快乐

# 列出所有情绪
emotion-dictionary list

# 搜索关键词
emotion-dictionary search 不安
emotion-dictionary search 爱

# 分析事件
emotion-dictionary analyze "今天工作压力很大，感觉快要崩溃了"
emotion-dictionary analyze "收到朋友的生日祝福，心里暖暖的"
```

## 命令参考

| 命令 | 描述 | 示例 |
|------|------|------|
| `query <情绪>` | 查询情绪详情 | `emotion-dictionary query 焦虑` |
| `list` | 列出所有情绪 | `emotion-dictionary list` |
| `search <关键词>` | 搜索情绪 | `emotion-dictionary search 不安` |
| `analyze <事件>` | 分析事件中的情绪 | `emotion-dictionary analyze "..."` |

## 理论基础

### 感受七层次

1. **原始感受** - 身体最直接的感受（冷、热、痛、痒）
2. **基本情绪** - 喜、怒、哀、惧、惊、厌恶
3. **情绪** - 焦虑、抑郁、孤独、嫉妒等
4. **情感** - 爱、感恩、信任、希望等
5. **公情** - 安全感、归属感、成就感等
6. **共情性情感** - 同理心、怜悯、感动等
7. **超越性情感** - 意义感、使命感、存在感等

### 回归疗法六步循环

1. **觉察** - 识别当下的感受
2. **接纳** - 允许感受存在，不评判
3. **探索** - 问自己"这个感受试图告诉我什么？"
4. **理解** - 找到背后的需求和欲望
5. **行动** - 采取小步骤满足需求
6. **调整** - 根据结果调整信念和策略

## 数据结构

每个情绪包含：
- 名称、拼音、分类、层次
- 定义、特征、身体反应
- 心理意义、应对策略
- 相关情绪、示例场景
