### 为什么需要 Gitee？

| 平台 | 国内访问 | 国际访问 | 建议用途 |
|------|----------|----------|----------|
| GitHub | ⚠️ 有时慢 | ✅ 快 | 国际用户 |
| Gitee | ✅ 秒开 | ⚠️ 慢 | 国内用户 |

### 3 步完成 Gitee 同步

**步骤 1**: 访问 https://gitee.com/new

**步骤 2**: 创建仓库
- 仓库名称：`emotion-dictionary`
- 仓库路径：`johnyang/emotion-dictionary`
- ❌ **不要勾选**"初始化仓库"

**步骤 3**: 推送代码
```bash
cd ~/.openclaw/workspace/skills/emotion-dictionary
git remote set-url gitee git@gitee.com:johnyang/emotion-dictionary.git
git push -u gitee main
```

如果需要 SSH key：
```bash
cat ~/.ssh/id_ed25519.pub
# 复制输出，添加到 https://gitee.com/profile/sshkeys
```
